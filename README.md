# Caleb Crouch's Portfolio

## Description

This is a personal portfolio. This will be used to show potential employers the skills I can bring to their business. There is a "Contact Me" section so employers can contact me, an "About Me" section so employers know who they would be hiring, and a "Work" section so they can see what I can do. The navigation bar, contact me section, and work section all have working links so the content I want people to see is easily reachable.

## Usage

To use this website you can either use the scroll function to scroll through the web page or use the Navigation Bar at the top to jump to the sections you would like to. Also, in the contact me section you can use the GitLab link and the Linkedin link to go to my personal GitLab and Linkedin Profile. All are included in the picture below.
